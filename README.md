## Git y Gitflow

Al ingresar Source Control puedo ver la diferencia de las versiones.
Se puede guardar cada cambio por separado.
Luego hacemos el commit.
 Si el archivo ya existe y esta en el repo
 - git commit -am "mensaje del commit"
    - a: add
    - m: message
Si el archivo no existe 
 - git add <nombre del archivo> o Git add . (agrega todo)
